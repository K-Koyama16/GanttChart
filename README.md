# Gantt Chart

### Package Version
- plotly 4.10.0
- streamlit 0.85.1

### How to use
1. Install plotly and streamlit
2. Enter the following command in a terminal
```
streamlit run gantt_sample.py
```

### Function
- Add tasks on the Gantt chart in response to T_DISPATCHED
